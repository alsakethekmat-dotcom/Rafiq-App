# تطبيق رفيق (Rafiq App) 🎓

"لأن كل طفلة تستحق وقتاً آمناً ومفيداً داخل مدرستها."

## نبذة عن المشروع

تطبيق **رفيق** هو نظام مدرسي ذكي مبتكر يهدف إلى تنظيم وقت انتظار الطالبات داخل **مدرسة حكمت الساكت الأساسية المختلطة**. يوفر التطبيق واجهة تفاعلية تتيح للطالبات حجز جلسات تعليمية وترفيهية هادفة مع مرشدات (طالبات)، مما يضمن استثمار وقت الفراغ بأسلوب آمن ومنظم.

## أهداف التطبيق

* تنظيم أوقات الانتظار المدرسي بفعالية.
* تعزيز روح التعاون بين طالبات المراحل الدراسية المختلفة.
* تقديم خدمات توجيهية متنوعة (حل واجبات، لعب موجه، متابعة دراسية).
* خلق بيئة مدرسية تفاعلية وآمنة.

## فريق العمل

هذا المشروع ثمرة جهود طالبات مدرسة حكمت الساكت الأساسية المختلطة:

* **الطالبات:** آلاء، جنى، سهام، ميار، سارة.
* **إشراف:** المعلمة أماني أبو رمان.

## المميزات التقنية

* **التصميم:** واجهة مستخدم (UI) متجاوبة مع كافة الشاشات (كمبيوتر وهاتف).
* **الهوية البصرية:** تصميم مستوحى من اللوغو الخاص بالمشروع، يعكس بيئة تعليمية إيجابية.
* **نظام التقييم:** إمكانية تقييم الخدمات المقدمة لضمان الجودة المستمرة.
* **سهولة الاستخدام:** نظام تنقل بسيط يتيح للطالبة الوصول للخدمة في خطوات معدودة.
